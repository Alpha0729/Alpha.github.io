DOM操作以及使用

1.nodeName属性可依据节点的类型返回其名称

            例如：oDiv.onclick=function(){
                    var node =oDiv.nodeName;
                    node=DIV   //返回出来的节点是大写的,如果需要利用toLowerCase转换成小写;
                }
            javaScript用法:object.nodeName;

2.添加或删除DOM元素

        2.1 、创建元素
        < div id="div1">
            < p id="p1">这是一个段落。< /p>
            < p id="p2">这是另一个段落。< /p>
        < /div>
        //创建一个元素p节点
        var para=document.createElement("p");
        //创建一个文本节点,目的是要往创建的元素里面添加
        var node=document.createTextNode("这是一个新段落。");
        //往新元素 p 里面添加内容
        para.appendChild(node);
        //获取元素div1
        var element=document.getElementById("div1");
        //最后添加元素到element里面
        element.appendChild(para);
        2.2 、删除元素
            var parent=document.getElementById("div1");
            var child=document.getElementById("p1");
            //删除parent里面的元素；
            parent.removeChild(child);
