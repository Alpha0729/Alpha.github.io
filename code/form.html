form表单相关属性

1、HTML < form>标签： 

        < form> 标签用于创建供用户输入的 HTML 表单,用法:
        < form action="www.baidu.com" method="get" target="blank" name="myName">
                action需要跳转的地址;
                method:规定用于发送表单数据的 HTTP 方法
                target:打开方式,blank在新的窗口打开,默认是当前窗口
                name： 属性规定表单的名称。
           //执行的表单
        < /form>
