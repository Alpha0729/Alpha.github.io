移动端常用语法

添加下面标签页使面自由缩放：

< meta name="viewport" content="width=device-width, initial-scale=1">

用JS使页面font-size：按比例显示：

            (function(win,doc){
                //  resize就是onresize事件;  当页面窗口没改变时触发该事件
                win.addEventListener('resize',change,false);
                function change(){
                    //以320屏幕为基础  也就是屏幕的宽度乘以20  是html的初始字体大小
                    doc.documentElement.style.fontSize=doc.documentElement.(clientWidth/320)*20+'px';
                    //**假如说屏幕宽度刚好是320px，该方法的font-size刚好是20px;如果说窗口大小不是320,
                    上面有个meta标签它的作用是等比例缩放,刚好达到自适应效果,何乐而不为呢;
             }
                change();
            })(window,document)

            *****使用rem的时候移动要除以20  因为默认字体是20px;

什么是rem：

W3C官网描述是“font size of the root element”，即rem是相对于根元素, 我们只需要在根元素确定一个参考值，在根元素中设置多大的字体，这完全可以根据您自己的需.
